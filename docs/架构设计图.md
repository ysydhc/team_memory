## 2. 核心数据流（四条路径）

```mermaid
flowchart LR
    subgraph flow1["① memory_recall / 检索"]
        direction TB
        r1["memory_recall"] --> r2["ExperienceService.search"]
        r2 --> r3["SearchPipeline"]
        r3 --> r4["向量 + FTS → RRF → 重排"]
        r4 --> r5["返回经验列表"]
    end

    subgraph flow2["② memory_save · project/personal"]
        direction TB
        s1["memory_save"] --> s2{路由}
        s2 -->|title + problem| s3["ExperienceService.save<br/>去重 · 向量 · 落库"]
        s2 -->|长 content| s4["LLMParser 抽取"]
        s4 --> s3
    end

    subgraph flow3["③ 档案创建（非 MCP scope=archive）"]
        direction TB
        a1["POST /api/v1/archives<br/>或 /archive skill"] --> a2["ArchiveService.archive_save<br/>文本 + 可选附件元数据 path/snippet<br/>（MCP memory_save 不支持 scope=archive）"]
        a2 --> a3["ArchiveRepository<br/>档案 + links + attachments 行"]
    end

    subgraph flow4["④ memory_context"]
        direction TB
        c1["memory_context"] --> c2["PersonalMemory 画像"]
        c1 --> c3["相关经验搜索"]
        c2 --> c4["profile + experiences"]
        c3 --> c4
    end

    style flow1 fill:#e5dbff,stroke:#5f3dc4
    style flow2 fill:#d3f9d8,stroke:#2f9e44
    style flow3 fill:#ffe8cc,stroke:#d9480f
    style flow4 fill:#e7f5ff,stroke:#1971c2
```

## 3. 档案馆：MCP 文本 vs HTTP 附件落盘

```mermaid
flowchart TB
    subgraph mcp_path["MCP（memory_* / 档案另走 HTTP POST /archives）"]
        M1["标题 + solution_doc 等文本"] --> M2["attachments 可选：path / snippet /<br/>content_snapshot 等元数据"]
        M2 --> M3["仅写 DB：archive_attachments 行<br/>不读本地文件、不写 uploads 目录"]
    end

    subgraph http_path["HTTP（Web / curl multipart）"]
        H0["须已有 archive_id"] --> H1["POST .../archives/{id}/attachments/upload"]
        H1 --> H2["读 file 字节"]
        H2 --> H3["uploads.root_dir/{archive_id}/<br/>.part → replace 终文件"]
        H3 --> H4["INSERT archive_attachments<br/>path = 相对 root 的 posix"]
        H4 --> H5["GET .../attachments/{att_id}/file 下载"]
    end

    subgraph fail["业务失败（已过 L2）"]
        F1["413 / 503 / 校验等"] --> F2["archive_upload_failures 可选记录"]
        F2 --> F3["Web 展示 + curl 模板补救"]
    end

    mcp_path -->|"创建档案得 id"| http_path
    http_path -.-> fail

    style mcp_path fill:#e7f5ff,stroke:#1971c2
    style http_path fill:#d3f9d8,stroke:#2f9e44
    style fail fill:#ffe3e3,stroke:#c92a2a
```
