## 二、亮点与可改进点

### 2.1 亮点

| 亮点 | 说明 |
|------|------|
| **需求澄清前置** | 入口形式（A/B/C）、主导航、高级配置位置逐项确认，避免返工 |
| **入口迁移干净** | 删除主导航、路由分支；Modal 首次打开时动态加载 workflow-viewer.js，避免首屏冗余 |
| **调试日志可开关** | _wfLog 支持 URL ?wf_debug=1 或 window._workflowDebug，便于排查 |
| **布局降级链** | Dagre → Cola → fCoSE，CDN 不可用时仍可渲染 |

### 2.2 可改进点

| 优先级 | 问题 | 建议 |
|--------|------|------|
| P3 | 详情按钮/点击曾多次不生效 | 坐标系统（renderedPosition vs overlay）与 Cytoscape tap 事件需在实现时优先验证；可增加 E2E 用例覆盖 |
| P3 | 调试日志默认开启 | 生产环境可改为默认关闭，仅 URL 或 localStorage 开启 |
| P3 | 单文件 $ref 提示 | 可考虑在 drop zone 文案中提前说明 |
